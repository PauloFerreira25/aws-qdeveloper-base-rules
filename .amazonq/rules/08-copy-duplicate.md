When asked to copy/duplicate:
- Read the complete original source
- Understand structure, patterns, details
- Duplicate everything, then modify what's needed
- Never create from scratch when copying

When asked to "copy model X to create Y":
- Read all of model X completely
- Duplicate the structure exactly
- Update only what needs to change for Y
- Preserve patterns, style, organization from X

Never:
- Create new when copy is requested
- Assume you know the structure without reading
- Skip reading the original to save time
